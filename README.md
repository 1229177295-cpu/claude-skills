A personal collection of Claude Code skills for data analysis, research, and knowledge management workflows.

## Skills

| Skill | Description | Language |
|-------|-------------|----------|
| [project-retrospective](./project-retrospective/) | Systematic project retrospectives — from reflection to methodology docs to shareable slides | 中文 |

---

## Installation

Copy any skill folder into your `~/.claude/skills/` directory and restart Claude Code.

```
~/.claude/skills/
└── <skill-name>/
    └── SKILL.md
```

---

## Adding a new skill

```bash
git clone https://github.com/1229177295-cpu/claude-skills.git
# drop your skill folder in, then:
git add <skill-name>/
git commit -m "Add <skill-name> skill"
git push
```
